# Project Blueprint

## Overview

This project is a high-end SaaS application with a focus on lead scoring, analytics, and automation. The goal is to build a robust and scalable system that provides valuable insights into lead quality and automates the lead management process.

## Implemented Features

*   **Lead Scoring:** A dynamic lead scoring system that assigns points to leads based on predefined rules.
*   **Automatic Score Updates:** Lead scores are automatically updated when lead information changes or when scoring rules are modified.
*   **Queued Score Recalculation:** A queued job to recalculate lead scores in the background, ensuring the application remains responsive.
*   **Lead Segmentation:** Automatically categorize leads into segments (e.g., "Hot", "Warm", "Cold") based on their scores.
